Contact form - top

  • Needs jqueries "jquery.validate.js" and "jquery.form.js", these are already attached in html.
  • You need to do / give following two things, to make contact form work....
    • Your email id in submit-form.php file 
    •               
 $to='yourname@mail.com';
    • Give path / url of php file "submit-form.php" in html as form action.
      Here path / url is ---- http://yoursitename.com/php/submit-form.php.
    •
  • Function for jquery validation is given in custom.js. No need to make changes there. 
  •             
 $(function() {
 var v = $("#form").validate({
 submitHandler: function(form) {
 $(form).ajaxSubmit({
 target: "#result",
 clearForm: true
 });
 }
 });
 });
  • Upload all files. Form will work fine.

Other more info about contact form

How to change mail body?

It can be changed in 'submit-form.php' file.

              
 $body='You have got a message...like this your desired text here'."\n\n";

How to give fixed / particular subject name to receiving mail?

If you want fix subject name for received mail, in case where u are not including subject field in contact form, it can be changed in 'submit-form.php' file.

              
 $subject = 'Fixed / particular subject name can be written here';

If subject field is included, then it will be like this.

        
 $subject = $subject;

How to change success message (message displayed after send process complete)?

Message which displays as result on send process complete, can be changed in php file, at place shown below.

 
 if(mail($to, $subject, $body, $headers)) {
 die('Your desired result text here');
 } else {
  die('Error: Mail failed');
 }

Why I'm getting error ..."There was an error sending your email. Please try again"

Form do not work offline, you need to upload all necessary files first.

How to add extra fields in contact form?

  • Add field html.
  • Add php for it, as it is added for other fields.
  • Choose suitable validation class for input field. Those are give in 'jquery.validate.js' approx. line no. 260 - 280.

Nivo slider - top

  • Needs nivo slider css (nivo-slider.css) and jquery (jquery.nivo.slider.pack.js), these are already attached in html.
  • Images are listed in div id "slider" with class "nivoslider", these both are specified in custom.js
    •
  • Function for nivo slider is in custom.js. There you can change effects, animation speed etc.
    •     
 $('#slider').nivoSlider({
 effect:'random', // Specify sets like: 'fold,fade,sliceDown' or choose 'random' for mixed effects
 slices:15, // For slice animations
 boxCols: 8, // For box animations
 boxRows: 4, // For box animations
 animSpeed:500, // Slide transition speed
 pauseTime:5000, // How long each slide will show
 pauseOnHover:true,
 directionNav:true, // Next & Prev navigation
 directionNavHide:true, // Only show on hover
 controlNav:false // 1,2,3... navigation
 });

To replace slider with video

Just remove the div '#slider' (all html code shown above). Just place your video link there.

If you don't need slider elsewhere also, remove the css, jquery for slider and slider fuction from custom.js.

Cycle plugin for testimonial - top

  • Needs cycle plugin jquery (jquery.cycle.all.min.js), which is already attached in html.
  • Function for cycle plugin is in custom.js. There you can change transition type, speed etc.
    •     
 $('.testimonial').cycle({
 fx: 'fade', // choose your transition type, ex: fade, scrollUp, scrollLeft etc... 
 speed: 1000,
 timeout: 4000,  // milliseconds between slide transitions (0 to disable auto advance)
 cleartypeNoBg:   true, // set to true to disable extra cleartype fixing (leave false to force background color setting on slides)
 cleartype:!$.support.opacity,  // true if clearType corrections should be applied (for IE) 
 pause:  1
 });
